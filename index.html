üìã Cara Setup Google Cloud Console (Penting!):
  1. Buat Project:
    Buka Google Cloud Console ‚Üí New Project
  2. Aktifkan Google Drive API:
    APIs & Services ‚Üí Library ‚Üí Cari "Google Drive API" ‚Üí Enable
  3. Buat API Key:
    APIs & Services ‚Üí Credentials ‚Üí Create Credentials ‚Üí API Key
    Copy API Key tersebut
  4. Buat OAuth 2.0 Client ID:
    Credentials ‚Üí Create Credentials ‚Üí OAuth client ID
    • Application type: Web application
    • Authorized JavaScript origins:
      http://localhost:8000 atau domain Anda
    • Authorized redirect URIs:
      http://localhost:8000 atau domain Anda
    Copy Client ID tersebut
  5. ‚ö†Ô∏è Konfigurasi OAuth Consent Screen (INI YANG PENTING!):
    APIs & Services ‚Üí OAuth consent screen
    • User Type: Pilih "External" lalu Create
    • App name: Isi nama aplikasi Anda
    • User support email: Isi email Anda
    • Developer contact: Isi email Anda
    • Klik Save and Continue
    • Scopes: Klik "Add or Remove Scopes"
      Cari dan tambahkan: ../auth/drive.file
    • Test users: Tambahkan email Google Anda sendiri sebagai test user!
    • Klik Save and Continue sampai selesai
  6. Paste Credentials:
    Copy Client ID dan API Key ke form di atas
